Star Plot¶

Star Plot is a little application to visualize the star field around a given attitude.

The star_plot module provides StarPlot a Qt widget to display a star field.

Example usage:

>>> from star_plot import StarPlot
>>> from Quaternion import Quat
>>> from Chandra.Time import DateTime

>>> w = StarPlot()
>>> quaternion = Quat(q=[-0.474674, -0.473931, 0.262471, 0.693674])
>>> starcat_time = DateTime('2020:001:19:18:21.914')
>>> w.show_stars(starcat_time, quaternion)
>>> w.show()

star_plot script¶

Plot the star field at a given time and around a given attitude

usage: star_plot [-h] [--obsid OBSID] [--date DATE]

Named Arguments¶

--obsid

OBSID

--date

Date in any CxcTime-compatible format
